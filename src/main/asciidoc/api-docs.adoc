[[overview]]
= Overview

[[overview-http-verbs]]
== HTTP method
|===
| Method | Usage

| `GET`
| 리소스 조회

| `POST`
| 리소스 생성 및 리소스 명령

| `PUT`
| 리소스 수정

| `DELETE`
| 리소스 삭제
|===

[[overview-http-status-codes]]
== HTTP status codes
|===
| Status code | Usage

| `200 OK`
| 요청 성공

| `201 Created`
| 생성 완료

| `400 Bad Request`
| 잘못된 요청

| `401 Unauthorized`
| 인증 실패

| `404 Not Found`
| 존재하지 않는 요청
|===
